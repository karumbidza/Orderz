## SKU Code Pattern Guide

Understanding the pattern helps you find the right code:

```
[CATEGORY]-[PRODUCT]-[SIZE/VARIANT]

Examples:
UNI-SHI-CAS-L = Uniform - Shirt - Cashier - Large
UNI-TRO-32   = Uniform - Trousers - Size 32
FTW-SHO-9    = Footwear - Shoes - Size 9
PPE-HARD-HAT = PPE - Hard Hat
STA-TILL-ROLL = Stationery - Till Roll
```

### Category Prefixes:
- `UNI-` = Uniforms
- `FTW-` = Footwear
- `PPE-` = Personal Protective Equipment
- `STA-` = Stationery
- `CON-` = Consumables
- `HDW-` = Headwear
- `OUT-` = Outerwear
- `FA-` = First Aid

### Role Codes (for Uniforms):
- `CAS` = Cashier
- `MGR` = Manager
